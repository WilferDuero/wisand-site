# wisand-site

Landing page de marketing para presentar **WISAND** como sistema de gestión para negocios físicos (inventario, ventas y control remoto).

## Stack

- HTML5
- CSS3
- JavaScript vanilla

## Estructura

```text
wisand-site/
├─ index.html
├─ css/
│  └─ styles.css
├─ js/
│  └─ main.js
└─ assets/
   ├─ placeholder-dashboard.svg
   ├─ icon-stock.svg
   ├─ icon-sales.svg
   └─ icon-remote.svg
```

## Uso local

1. Abre `wisand-site/index.html` en tu navegador.
2. Si quieres cambiar el WhatsApp, edita `WHATSAPP_NUMBER` en `wisand-site/js/main.js`.

## Deploy en Vercel

1. Crea un nuevo proyecto en Vercel.
2. Selecciona la carpeta `wisand-site`.
3. Deploy sin configuración extra (sitio estático).

## Deploy en Netlify

1. En Netlify, crea un nuevo sitio.
2. Sube la carpeta `wisand-site` como deploy manual o conecta el repo.
3. Publica sin build command (sitio estático).
